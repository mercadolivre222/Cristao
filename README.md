# PlayKids Cristão

🎮 **Página de vendas do PlayKids Cristão** - 10 Jogos Bíblicos que Ensinam Valores Enquanto seu Filho se Diverte no Celular!

## 🚀 Como começar

### Instalação

```bash
npm install
```

### Desenvolvimento

```bash
npm run dev
```

A aplicação abrirá em `http://localhost:3000`

### Build para Produção

```bash
npm run build
```

Os arquivos de produção serão gerados na pasta `dist/`.

## 📦 Deploy na Vercel

1. Faça push do código para o GitHub
2. Vá para [vercel.com](https://vercel.com)
3. Clique em "New Project"
4. Selecione o repositório `mercadolivre222/Cristao`
5. Configure:
   - **Framework**: Vite
   - **Build Command**: `npm run build`
   - **Output Directory**: `dist`
6. Clique em "Deploy"

## 📄 Estrutura do Projeto

```
├── index.html           # HTML principal
├── vite.config.js       # Configuração do Vite
├── package.json         # Dependências
├── src/
```
